¿Qué es GitHub?

GitHub es una plataforma en línea para alojar y colaborar en proyectos de software. Utiliza Git, que es un sistema de control de versiones. Google Drive y LinkedIn, pero para desarrolladores. En pocas palabras:

  • Git → Herramienta para llevar el historial de cambios de tu proyecto.
  • GitHub → Sitio web para guardar esos proyectos y trabajar con otras personas fácilmente.

🔑 ¿Para qué se usa?

  • Guardar y respaldar tu código en la nube.
  • Colaborar con otros desarrolladores (incluso millones de desconocidos).
  • Revisar y discutir cambios en el código (pull requests).
  • Organizar tareas, errores y mejoras (issues).
  • Automatizar pruebas y despliegues (GitHub Actions).
  • Crear páginas web estáticas (GitHub Pages).

🗂️ Elementos clave

  • Repositorio (Repository): El contenedor de tu proyecto (archivos, historial de cambios, documentación).
  • Branch (Rama): Una línea de trabajo paralela al código principal. Sirve para desarrollar nuevas funciones sin afectar el proyecto estable.
  • Commit: Un «snapshot» de los cambios que has hecho.
  • Pull Request (PR): Una solicitud para fusionar cambios de una rama a otra, normalmente de una rama de trabajo a la rama principal.
  • Fork: Una copia de un repositorio ajeno para proponer cambios sin afectar el original.
  • Issue: Un ticket para reportar errores, sugerir mejoras o planificar tareas.

👤 Tu perfil de GitHub

Tu perfil se encuentra en https://github.com/tu-usuario. Es como tu currículum online. Los reclutadores y otros desarrolladores pueden ver tus proyectos y actividad.

⭐️ Estrellas, forks y comunidad

  • ⭐️ Estrella: Marca un proyecto como favorito para guardarlo.
  • 🍴 Fork: Crea tu propia copia de un repositorio para modificarlo.
  • 👥 Comunidad: Puedes seguir personas, comentar código y colaborar.

🎯 ¿Por qué es importante?

  • Es tu portafolio profesional.
  • Permite mostrar tus habilidades y proyectos.
  • Te conecta con la comunidad open source.

🌐 ¿Qué es HTML?

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje básico de la web y se usa para estructurar el contenido de las páginas.

HTML funciona como el esqueleto de una página web. Describe qué elementos hay (texto, imágenes, enlaces), pero no cómo se ven ni cómo funcionan de forma dinámica.

🏗️ ¿Cómo funciona?

HTML usa etiquetas (tags) para marcar cada parte del contenido. Ejemplos comunes:

  • <h1> → Título principal
  • <p> → Párrafo
  • <a> → Enlace
  • <img> → Imagen
  • <div> → Contenedor

📄 Estructura básica

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>¡Hola, mundo!</h1>
    <p>Bienvenido a mi página web.</p>
  </body>
</html>

🔑 Características principales

  • ✅ Fácil de aprender
  • ✅ Es la base del desarrollo web
  • ✅ Compatible con todos los navegadores
  • ✅ Permite incrustar multimedia, formularios, tablas y más

🗂️ Relación con otros lenguajes

  • HTML: Estructura
  • CSS: Estilos
  • JavaScript: Comportamiento

🚀 ¿Qué puedes hacer con HTML?

  • Crear páginas web simples
  • Diseñar formularios de contacto
  • Construir bases de blogs, portafolios, tiendas en línea
  • Integrar contenido multimedia (videos, mapas)

✨ Ejemplo práctico

Aquí tienes un ejemplo de un párrafo, un enlace y una imagen:

<p>Este es un párrafo de ejemplo.</p>

🎨 ¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje usado para dar estilos y presentación a una página web creada con HTML.

📌 ¿Para qué sirve?

  • Definir colores, fuentes y tamaños de texto.
  • Crear márgenes y rellenos (padding).
  • Organizar elementos en columnas o grillas.
  • Agregar efectos como sombras, transiciones o animaciones.
  • Adaptar el diseño a diferentes dispositivos (responsive design).

💡 Ejemplo básico:

h1 {
  color: blue;
  font-family: Arial, sans-serif;
  text-align: center;
}

p {
  color: #555;
  font-size: 1.2rem;
}

🎯 Reglas CSS: Selector + Propiedad + Valor

Una regla CSS se compone de:

  • Selector: el elemento al que se aplica (por ejemplo, h1).
  • Propiedades: las características que quieres cambiar (color, fuente, margen).
  • Valores: el valor de cada propiedad (blue, Arial, 20px).

🔑 Buenas prácticas:

  • Usa clases para reutilizar estilos.
  • Organiza tu CSS en un archivo separado (por ejemplo, style.css).
  • Combina CSS con frameworks como Bootstrap para acelerar tu diseño.

📐 ¿Qué es Flexbox?

Flexbox es un módulo de diseño de CSS que facilita alinear, distribuir y ordenar elementos dentro de un contenedor flexible.

📌 ¿Por qué usar Flexbox?

  • Alinea elementos horizontal o verticalmente de forma sencilla.
  • Distribuye espacio entre elementos de forma automática.
  • Hace que el diseño sea flexible y responsivo.

💡 Ejemplo básico:

.contenedor {
  display: flex;
  justify-content: center;
  gap: 1rem;
}

.item {
  background: #0d6efd;
  color: white;
  padding: 1rem 2rem;
  border-radius: 0.5rem;
}

🎯 Propiedades clave de Flexbox

  • display: flex; → Activa Flexbox en el contenedor.
  • flex-direction → Define la dirección de los ítems (row, column).
  • justify-content → Alinea ítems horizontalmente (start, center, space-between).
  • align-items → Alinea ítems verticalmente (stretch, center).
  • gap → Espacio entre ítems.

📌 Ejemplo con dirección de columna:

.columna {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 1rem;
}

📐 Unidades de medida en CSS

En CSS usamos unidades para definir tamaños, márgenes, anchos, alturas, fuentes, posiciones y más. Se dividen en dos grupos principales: absolutas y relativas.

1️⃣ Unidades Absolutas

Son fijas y no cambian según el contexto. Ejemplos:

  • px → píxeles (unidad más común en pantallas).
  • cm → centímetros.
  • mm → milímetros.
  • in → pulgadas.
  • pt → puntos (1pt = 1/72 in).
  • pc → picas (1pc = 12pt).

2️⃣ Unidades Relativas

Dependen de otras medidas como la fuente, el contenedor o el tamaño de la ventana:

  • % → porcentaje respecto al contenedor.
  • em → relativo a la fuente del elemento actual.
  • rem → relativo a la fuente raíz (<html>).
  • vw → 1% del ancho de la ventana.
  • vh → 1% de la altura de la ventana.
  • vmin → 1% del lado más pequeño de la ventana.
  • vmax → 1% del lado más grande de la ventana.
  • ch → ancho del carácter "0" de la fuente usada.
  • ex → altura de la letra "x" de la fuente.

💡 Ejemplo práctico

/* Absoluta */
.box {
  width: 300px;
  height: 200px;
}

/* Relativa al contenedor */
.container {
  width: 80%;
}

/* Relativa a la fuente */
.text {
  font-size: 2em;
}

.root-text {
  font-size: 1.5rem;
}

/* Relativa al viewport */
.hero {
  height: 100vh;
  width: 100vw;
}

✅ Consejos

  • Usa rem para fuentes escalables y consistentes.
  • Usa % para contenedores fluidos.
  • Usa vw y vh para diseños responsivos a pantalla completa.
