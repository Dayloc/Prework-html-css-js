¿Qué es GitHub?

GitHub es una plataforma online donde puedes guardar tu código, colaborar con otros y mostrar tus proyectos. Es como una mezcla entre Google Drive y LinkedIn, pero para desarrolladores.

📁 ¿Qué es un repositorio?

Un repositorio (o "repo") es como una carpeta de tu proyecto. Ahí puedes guardar todo tu código, imágenes, datos, etc.

  • Puedes tener un repo por cada proyecto.
  • Se puede compartir con otros.
  • Permite colaborar y controlar los cambios.

👤 Tu perfil de GitHub

Tu perfil se encuentra en https://github.com/tu-usuario. Es como tu currículum online. Los reclutadores y otros desarrolladores pueden ver tus proyectos y actividad.

⭐️ Estrellas, forks y comunidad

  • ⭐️ Estrella: Marca un proyecto como favorito para guardarlo.
  • 🍴 Fork: Crea tu propia copia de un repositorio para modificarlo.
  • 👥 Comunidad: Puedes seguir personas, comentar código y colaborar.

🎯 ¿Por qué es importante?

  • Es tu portafolio profesional.
  • Permite mostrar tus habilidades y proyectos.
  • Te conecta con la comunidad open source.

👉 Crea tu cuenta en GitHub.com y empieza a construir tu presencia como desarrollador.

¿Qué es HTML?

HTML es el lenguaje de marcado que usamos para estructurar el contenido de una página web.

<h1>Hola Mundo</h1>

Hola Mundo

¿Qué es CSS?

CSS se usa para dar estilo al contenido HTML.

h1 {
  color: blue;
  font-family: Arial;
}

Hola Mundo estilizado

¿Qué es Flexbox?

Flexbox es un sistema de diseño que permite alinear elementos fácilmente.

.contenedor {
  display: flex;
  justify-content: center;
  gap: 1rem;
}
